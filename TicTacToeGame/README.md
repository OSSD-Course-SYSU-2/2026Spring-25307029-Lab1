# 双人井字棋游戏（鸿蒙版）

## 游戏特色

这是一个适配鸿蒙系统的双人井字棋游戏，与传统井字棋不同，**棋子只会在棋盘上存在10秒**，这大大增加了游戏的策略性和可玩性。

## 游戏规则

1. **双人对战**：玩家1使用"X"，玩家2使用"O"
2. **轮流下棋**：双方轮流在空的格子上放置棋子
3. **时间限制**：每个棋子放置后会在10秒后自动消失
4. **获胜条件**：先将自己的三个棋子连成一条线（横、竖、斜）的玩家获胜
5. **平局判定**：如果棋盘被填满且无人获胜，则为平局

## 策略要点

- 由于棋子会消失，需要考虑时机和位置
- 可以利用棋子消失的特性创造新的连线机会
- 阻止对手的同时要考虑自己的棋子何时会消失
- 倒计时显示帮助玩家规划策略

## 技术实现

### 核心功能

1. **棋子生命周期管理**
   - 每个棋子记录放置时间戳
   - 定时器每秒更新剩余时间
   - 10秒后自动移除棋子

2. **游戏状态管理**
   - 当前玩家切换
   - 胜负判定算法
   - 分数统计

3. **UI交互**
   - 点击放置棋子
   - 实时倒计时显示
   - 游戏结果提示

### 关键代码说明

- `CellData` 接口：存储棋子的玩家信息、时间戳和剩余时间
- `updateBoard()` 方法：每秒检查并移除过期棋子
- `checkWinner()` 方法：检查行、列、对角线是否连成线
- 定时器管理：组件生命周期内启动和清理
